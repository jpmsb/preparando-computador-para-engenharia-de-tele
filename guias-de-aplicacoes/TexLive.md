## Uso

Após a instalação, um dos compiladores disponíveis é o `lualatex`, que pode ser usado conforme o exemplo abaixo:

```bash
lualatex -shell-escape -synctex=1 -interaction=nonstopmode -file-line-error -recorder  "monografia.tex"
```

A seguir, está o significado de cada argumento:

- `shell-escape`: Permite a execução de comandos do sistema operacional a partir do documento LaTeX.
- `synctex=1`: Habilita a sincronização entre o código fonte e o documento PDF gerado.
- `interaction=nonstopmode`: Faz com que o compilador continue a compilar mesmo se encontrar erros, sem pausar para interação do usuário.
- `file-line-error`: Fornece mensagens de erro mais detalhadas, incluindo o número da linha onde o erro ocorreu.
- `recorder`: Gera um arquivo `.fls` que lista todos os arquivos lidos e escritos durante a compilação, útil para ferramentas de construção.
- `"monografia.tex"`: Especifica o arquivo fonte LaTeX a ser compilado.
